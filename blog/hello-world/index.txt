13:T44d,<p>This is the first post on the new blog. It's built from a single Markdown

<code>remark</code>, and exported statically — no server, no database, no CMS.</p>
<h2>Why a blog</h2>
<p>I wanted a place to write short, informal notes on cloud security, IAM,
and whatever I'm building — separate from the curated <code>/writing</code> page,
which links out to longer external pieces.</p>
<h2>How it works</h2>
<p>Every post is a Markdown file with frontmatter:</p>
<pre><code class="language-md">---
title: "Hello, World"
date: "2026-07-03"
slug: "hello-world"
description: "The first post on this blog, and a quick note on why it exists."
---
</code></pre>
<p><code>generateStaticParams</code> reads every slug in <code>content/blog/</code> at build time,
so each post gets its own statically exported HTML page — no manual
build step, no rebuild trigger beyond a normal <code>git push</code>.</p>
<p>To publish a new post, add a new <code>.md</code> file here and push to <code>main</code>.
That's it.</p>
